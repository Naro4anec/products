## Установка

- Скопировать репозитой
- Установить зависимости 
```sh
composer install
```
- Подключить к БД. Прописать данные для входа в файл .env
- Выполнить миграцию
```sh
php artisan migrate
```
- Импортировать данные в БД или заполнить тестовыми. Будет добавлено 5 магазинов и 30 товаров.
```sh
php artisan db:seed
```
- Запустить
```sh
php artisan serve
```
